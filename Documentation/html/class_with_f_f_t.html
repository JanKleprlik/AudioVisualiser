Generates logarithmically scaled frequency halo forming a circle. Each layer is stored at one VertexArray in the vector VAs. Parameters: Color of each layer is in vector colors. Height of each layer is in vector heights. Center is the center of the halo. Radius is the base radius the halo will reach when no music is playing. From is a frequency from which the halo should start generating. To is a frequency to which the halo should be generated.
